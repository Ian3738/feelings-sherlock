# 做我的情緒偵探｜Feelings Sherlock（線上卡牌桌遊）

兒童情緒教育卡牌遊戲的數位版：50 張 AI 繪製情緒插圖卡、五大情緒家族。

🎮 **立即遊玩：https://ian3738.github.io/feelings-sherlock/**

## 遊戲模式
- 🌐 **線上房間・同感偵探**：2–4 人各自用手機／平板加入房間，猜猜主角的心情（Firebase 即時連線）
- 🎭 **表演猜猜樂**：一台裝置輪流傳，60 秒表演情緒大家搶答
- 🔍 **偵探任務**：情境 → 情緒線索 → 心情溫度計 → 因應行動的引導式練習
- 🃏 **情緒圖鑑**：50 種情緒 × 五大家族，附情緒地圖與心情溫度計

## 技術
單一 HTML 檔（插圖以 base64 內嵌）＋ Firebase Realtime Database（asia-southeast1）。
卡牌插圖由 Gemini（gemini-3.1-flash-image）生成。

※ 本遊戲玩法架構參考俐蓓爾《做我的情緒偵探》實體牌卡，插圖與程式為原創重製，供教育用途。
